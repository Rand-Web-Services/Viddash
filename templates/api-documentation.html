API Documentation

Viddash API access is designed for teams that need repeatable conversion, compression, and media preparation workflows.

Current API Status

The public API is being prepared for Business customers. The app already exposes internal conversion endpoints, and the production API will add authentication, rate limits, usage reporting, and stable response schemas.

Planned Capabilities

  • Image conversion and resizing jobs.
  • Video conversion, compression, resizing, and clip extraction.
  • Transcript and caption generation workflows.
  • Webhook callbacks for completed jobs.
  • Team-level quota, billing, and API key management.
